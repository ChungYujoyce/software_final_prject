Install VideoH
1. go to public_html/config/Config.php and replace FB config with yours.
2. go to public_html/config/MySQL.php and replace with your own settings.
3. go to public_html/views/js/login.js and replace client_id with yours in line 50
4. go to public_html/views/js/client.js and replace the url with yours in line 7
5. go to nodeServer/server.js and replace the port with yours in line 8
6. create the db by db.sql in public_html
